**Impact:** The ampOS database (Supabase) was unavailable during this window.

**Action required:** If you were doing work in ampOS during that window (e.g., creating or editing records, running reports, submitting forms), please **verify that your changes are still intact**. If anything is missing or looks incorrect, report it so we can track and address it.
