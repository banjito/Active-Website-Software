## 🔧 Usage

This folder contains guides for fixing issues that require manual intervention or don't have automated solutions.

## ⚠️ Before You Start

1. **Backup data** before making manual changes
2. **Read the entire guide** before starting
3. **Test in development** if possible
4. **Document your changes** for future reference

## 📝 When to Use

Use these guides when:
- Automated fixes aren't available
- Data needs manual correction
- Edge cases require special handling
- System state needs manual intervention
