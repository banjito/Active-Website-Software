# Database & Schema

Database structure documentation and schema diagrams.

## 📋 Contents

- `schema_diagram.dbml` - Database schema diagram (DBML format)
- `schema_relationships.md` - Table relationships and foreign keys

## 🗺️ Schema Overview

The schema documentation provides a visual and textual representation of the database structure, including:
- Table definitions
- Column types and constraints
- Relationships between tables
- Indexes and keys

## 📝 Usage

Use these files to understand the database structure when:
- Adding new tables or columns
- Creating migrations
- Planning features that require database changes
- Debugging data issues
