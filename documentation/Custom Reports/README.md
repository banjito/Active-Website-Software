## 🎯 Purpose

The Custom Form Builder allows users to create custom report templates with drag-and-drop functionality, incorporating data from any table (nameplate, fuse data, temperature correction, etc.).

## 🚀 Key Features

- Drag-and-drop form builder
- Reusable component library
- Temperature correction integration
- Save templates per user
- Share templates across organization
- Link to job assets
- PDF generation
